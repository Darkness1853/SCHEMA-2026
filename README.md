**Проблемы в ходе эксплутации:**

1. Необходимо менять файл .qsf если выдает ошибку с неверными пинами, если вы настраивали их, то  нужно будет заккоммитить в файле, чтоб не было ошибок.

2. ModelSim: перезапуск при создании нового тестбенча
После создания нового файла для тестирования (testbench) обязательно перезапустите ModelSim — в противном случае новый файл может не подключиться корректно.

3. ModelSim: перезапуск при ошибке записи кода
Если ModelSim не может записать в файл код (ошибка записи), это также лечится перезапуском среды.

4. После любого изменения кода → сохраняйте и перекомпилируйте
При каждом изменении исходного кода (в Quartus или ModelSim):

Сохраните файл (Ctrl+S).

Заново скомпилируйте проект в ModelSim, чтобы изменения вступили в силу.
